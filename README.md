# DataScript Compiler

A custom programming language compiler developed as an academic project. The compiler performs lexical analysis, syntax analysis, semantic analysis, and code generation through a simple web interface.

## Features

- Lexical Analysis (Tokenization)
- Syntax Analysis (Parser)
- Semantic Analysis
- Code Generation
- Simple Frontend Interface
- Python Backend

## Tech Stack

- Python
- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository

```bash
git clone https://github.com/anubhavsharma22/DataScript-Compile.git

2. Navigate to the project directory

```bash
cd DataScript-Compile
```

3. Install dependencies

```bash
pip install -r backend/requirements.txt
```

4. Run the application

```bash
python backend/app.py
```
